# 三周刷题计划

> 原则：**一天 2-3 题，新题配旧题复习。重在能独立默写 + 出声讲，不在数量。**
> 你的痛点是"看懂写不出"——所以每题都要**关答案默写**，写不出就用"卡住五步"。

## 第 1 周：巩固舒适区 + 补基础模板

**目标**：把你已会的题钉死，补齐"二分、链表反转、栈、BFS"四个必会模板。

- **Day 1**：复习 1, 121, 242(已会，限时 10 分钟/题默写) + 新 704(二分模板)
- **Day 2**：复习 217, 169 + 新 20(栈，Valid Parentheses)
- **Day 3**：新 206(反转链表，必会)+ 新 21(合并链表)
- **Day 4**：复习 3(滑窗)+ 新 53(Maximum Subarray / Kadane)🔥
- **Day 5**：新 102(二叉树层序遍历 BFS 模板)🔥 + 复习 200
- **Day 6**：新 35, 153(二分变体)
- **Day 7**：**休息或补做没完成的**。回顾这周所有题，挑 2 道再默写一遍。

## 第 2 周：攻高频中等题(Google 最爱)

**目标**：拿下频率榜上的 Med 重点题。

- **Day 8**：15(3Sum)🔥 —— 排序+双指针，**Google 超高频**，多花时间
- **Day 9**：49(Group Anagrams)🔥 + 复习 242
- **Day 10**：33(Rotated Sorted Array)🔥 + 复习 704

- **Day 12**：322(Coin Change)🔥 —— DP 重点
- **Day 13**：22(Generate Parentheses)🔥 —— 回溯模板
- **Day 14**：**休息/补做**。复习本周 DP 三题(70/198/322)。

## 第 3 周：设计题 + 巩固 + 模拟

**目标**：补设计类大题，整体串一遍，进入面试状态。

- **Day 15**：146(LRU Cache)🔥 —— Google 经典设计题，**重点**

- **Day 17**：560 默写(前缀和)+ 567(滑窗)

- **Day 20**：**模拟面试**：随机抽 2 道做过的题，全程出声讲英文 + 限时 25 分钟
- **Day 21**：**休息/查漏**。把最不熟的 3 道重做。

# 你的专属注意事项

1. **每题先口头答两问**：用 set 还是 dict？遍历用下标还是取值？(你最近的判断口诀)
2. **关答案默写**，不是看懂就过。写不出 → "说出卡点 → 写暴力 → 举例手推 → 要提示 → 讲思路"。
3. **Hard 题(76, 239, 42)只求看懂思路 + 能写暴力**，不强求独立写最优解。
4. **Java 转 Python 省力工具**记牢：Counter / heapq.nlargest / sorted(key=) / set 运算 / "".join / str(n) 拆位。
5. **每道题练完出声讲一遍英文**(复述题意→暴力→优化→复杂度)，这是面试真正考的。
